一个模组，为 Minecraft 增加一堆非常逆天的东西。

## 附魔一览

### 🏹 弓 / 弩通用

| 附魔 | 适用物品 | 最高等级 | 说明 |
|------|---------|---------|------|
| 精确射手 (Precise Shooter) | 弓、弩 | III | 提高弹道精度并增加伤害倍率。每级降低 20% 散布，伤害倍率 +0.75× |

### 🔫 弩专属

| 附魔 | 最高等级 | 说明 |
|------|---------|------|
| 自动机 (Automatic) | V | 按住使用键自动连射弩箭，射速随等级提升（4/6/8/10/12 发/秒）。与三连发互斥 |
| 三连发 (Triple Burst) | III | 单次射击后自动追加 2 发点射，间隔随等级缩短（5/4/3 tick）。与多重射击和自动机互斥 |
| 爆破烟花 (Blast Firework) | V | 增强弩发射的烟花火箭爆炸效果。与潜影烟花互斥 |
| 潜影烟花 (Shulker Firework) | VI | 烟花火箭命中后生成探头贝（Peekaboo Shell）实体进行追踪攻击。与爆破烟花互斥 |
| 铁傀儡烟花 (Golem Firework) | I | 烟花火箭命中后产生铁傀儡相关效果。与苦力怕烟花互斥 |
| 苦力怕烟花 (Creeper Firework) | I | 烟花火箭命中后产生苦力怕爆炸效果。与铁傀儡烟花互斥 |
| 精确制导 (Precision Guidance) | I | 弩发射的烟花火箭具备制导能力，优先追踪被「被锁定」效果标记的目标，无锁定目标时搜索视场圆锥内的敌对生物。支持发射后动态切换目标 |

### 🔱 三叉戟

| 附魔 | 最高等级 | 说明 |
|------|---------|------|
| 轰顶 (Sky Bombard) | IV | 三叉戟投掷后产生空中轰炸效果。与激流互斥 |

### 🪖 头盔

|------|---------|------|
| 热成像头盔 (Thermal Helmet) | III | 夜间自动在范围内的温血生物身上显示发光粒子效果，范围为 (6 + 等级) × 16 格。排除冷血/水生/机械生物 |
| 鱿鱼铁拳 (Squid Iron Fist) | V | 自动检测来袭的敌对弹射物和飞行怪物，发射缩小版发光鱿鱼进行拦截。命中后产生爆炸、火焰和墨汁效果，附带前向溅射伤害 |

|------|---------|------|

### 🐐 山羊角

### 🪄 木棍

| 附魔 | 最高等级 | 说明 |
|------|---------|------|
| 奇异法杖 (Strange Wand) | I | 右键点击敌对生物有 50% 概率将其变形为同体型分类的其他怪物（飞行/水生/四足/双足）。右键点击村民可治疗并有 20% 概率使其变为幼年。仅在主世界生效，3 秒冷却 |

## 状态效果

| 效果 | 说明 |
|------|------|
| 被锁定 (Locked On) | 标记目标为制导弹药的追踪对象，同时附带发光效果和烟花粒子轨迹 |
| 反应装甲冷却 (Reactive Armor Cooldown) | 反应装甲触发后的冷却期指示 |
| 鱿鱼铁拳冷却 (Squid Iron Fist Cooldown) | 鱿鱼铁拳触发后的冷却期指示 |

## 自定义物品

| 物品 | 说明 |
|------|------|
| 目标指针 (Target Pointer) | 指南针类物品，自动指向最近的被「被锁定」效果标记的目标，每秒刷新一次 |
| 探头贝刷怪蛋 (Peekaboo Shell Spawn Egg) | 用于生成探头贝实体的刷怪蛋 |

所有自定义物品和附魔书均可在创造模式「逆天附魔」物品栏标签页中找到。

## 自定义实体

| 实体 | 说明 |
|------|------|
| 探头贝 (Peekaboo Shell) | 潜影烟花附魔生成的潜影贝变体，具有追踪攻击能力 |
| 探头贝火花 (Peekaboo Spark) | 探头贝发射的追踪弹射物 |

## 附魔互斥关系

- 自动机 ↔ 三连发
- 爆破烟花 ↔ 潜影烟花
- 铁傀儡烟花 ↔ 苦力怕烟花
- 轰顶 ↔ 激流（原版）
- 三连发 ↔ 多重射击（原版）

## 安装

1. 安装 [Fabric Loader](https://fabricmc.net/) ≥ 0.15.0
2. 安装 [Fabric API](https://modrinth.com/mod/fabric-api)
3. 将模组 jar 文件放入 `.minecraft/mods` 目录
4. 启动游戏

## 构建

```bash
./gradlew build
```

构建产物位于 `build/libs/` 目录。
